How to play

One player hosts a game and shares the 6-letter code (or QR code). Everyone else opens the app, taps Join, and enters the code and a name — no account needed. Each player gets a secret role; talk it out, vote, and find the Vampire before time runs out.

Hosting & purchases

Joining a game is always free. Hosting starts with a free trial, then a one-time unlock keeps hosting forever — no subscription. Only one person per group needs to unlock hosting.

Restore a purchase

New phone or reinstalled the app? Open the paywall and tap Restore Purchases to bring back your hosting unlock.

FAQ

Can players see each other? Make sure everyone joins the same game code.

"Name already taken"? Names are unique within a game — pick a different one.
